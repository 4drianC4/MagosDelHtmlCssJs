Esta lista contiene algunos proyectos para practicar y aumentar más su nivel.

# Low Level (Prácticas que hicimos y algunos extras)

## Mi Carta de Hechicero/a

Una página personal con tu foto, descripción, hobbies y poderes (habilidades), usando solo HTML y CSS.

## Tarjetas de personajes mágicos

Crea varias tarjetas con nombre, imagen, descripción y habilidades, puedes usar flexbox y grid y algunas animaciones interactivas para cards con javascript y css.

## Formulario de ingreso al gremio de magos

Un formulario completo con validaciones básicas (HTML5), estilos personalizados y diseño limpio.

# Página de historia mágica (Landing Page)

Una página con títulos, secciones, imágenes, testimonios de otros magos y un llamado a la acción.

## Horario encantado

Tabla de horarios con diferentes estilos por fila o columna. Ideal para practicar table, :hover, nth-child, etc. debes ingresar por javascript tus datos.

# Low-Mid Level

## Calculadora mágica

Usa funciones JS para realizar operaciones matemáticas con diseño personalizado.

## Simulador de pociones

Ingresar ingredientes (texto) y al hacer clic mezclar y mostrar el resultado final (manejo de listas, según a cada combinación dar una respuesta diferente).

## To-Do list de misiones mágicas

Agregar, tachar, eliminar y guardar tareas. Uso del DOM y eventos (addEventListener, createElement, etc.).

## Formulario interactivo con DOM

Valida edad, nombre, y muestra una respuesta personalizada con efectos visuales.

## Página de perfil con modo oscuro

Un solo botón cambia completamente la apariencia de la web.

  

# Mid Level (Saliendo del contexto y siendo más serios)

## Blog Personal con CRUD

Crear, leer, actualizar y borrar artículos. Manejo del DOM, almacenamiento local (localStorage) o integración con una API ficticia. Agrega un panel de administración.

  

## Clon de una Página Popular

Clon de Netflix, YouTube, Google, Twitter. Replicar diseño y funcionalidades clave. Requiere atención al detalle, componentes dinámicos, sliders, popups, etc.

  

## To-Do App Avanzada

Lista de tareas con categorías, fechas, búsqueda y filtros. Manejo avanzado del DOM y eventos, estado, almacenamiento persistente.

## Dashboard de Administración de Datos

Tablero con estadísticas, gráficos, filtros y formularios. Tecnologías sugeridas: Chart.js o ApexCharts. Responsive + modo oscuro.

## Clon de E-commerce

Página de productos, carrito, checkout (simulado). Manipulación de datos, validación de formularios, diseño modular. Filtrado y búsqueda dinámica.

## Portafolio Interactivo

Objetivo: Mostrar tus proyectos con animaciones y efectos visuales. Diseño responsivo, animaciones con CSS o JS (GSAP o ScrollReveal). Modo oscuro, formulario de contacto funcional.

# Juego en el Navegador

Ejemplos: Ahorcado, pocker, 21, Snake, tres en raya, cacho. Aplicar lógica de programación, eventos y animaciones. Agrega niveles, puntuación y persistencia con localStorage.

## Conversor de Unidades

Moneda, temperatura, longitud. Buena lógica, validación de formularios, diseño limpio.

## Sitio de Noticias con API

Usa la API de NewsAPI o similar para mostrar noticias actualizadas, filtrado por categoría o país. Agrega paginación, loading spinner y diseño adaptable.

## Página con Autenticación Simulada

Crear login/register con JS simulando backend. Validación, manejo de usuarios en localStorage, control de sesiones.

# Hard Level

## Single Page Application (SPA) sin frameworks

Implementa tu propio enrutador con history.pushState y popstate. Simular navegación entre páginas sin recargar. Agrega un sistema de rutas dinámicas y carga condicional de componentes.

## Editor de Texto Online (como Notion o Google Docs básico)

Edición de texto enriquecido con contentEditable. Manejo de comandos de edición, barras de herramientas dinámicas. Exportar a HTML o Markdown.

## Kanban Board tipo Trello

Arrastrar y soltar tareas entre columnas. drag and drop API o bibliotecas personalizadas. Guardado en localStorage o sincronizado con Firebase.

## Constructor de Formularios Dinámico

El usuario arrastra y suelta campos de formulario y los configura. Guarda el JSON de la estructura creada y permite renderizarla.

## Dashboard con WebSockets (en tiempo real)

Usa Socket.io en backend y frontend (requiere Node.js). Chat en tiempo real, actualizaciones en vivo de datos. Autenticación de usuarios y notificaciones en pantalla.
