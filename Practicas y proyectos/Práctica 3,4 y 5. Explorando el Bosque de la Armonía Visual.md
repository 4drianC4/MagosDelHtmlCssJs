# Parte 1: La Elección de tu Aura Visual (Investigación)

Antes de aplicar hechizos CSS, necesitas definir tu identidad mágica visual:
## Paleta de Colores:
1. Investiga páginas como coolors.co o colorhunt.co
2. Elige una paleta de 3 a 5 colores que represente tu estilo mágico.
3. Justifica por qué la elegiste (calidez, energía, serenidad, oscuridad, etc.).

## Tipografía:
1. Visita Google Fonts.
2. Elige una tipografía principal para títulos y una secundaria para texto normal.
3. Integra ambas en tu proyecto y aplica con las propiedades font-family.
## Layout:
1. Decide si usarás Flexbox o Grid para organizar tus secciones.
2. Boceta o describe cómo se verán tus secciones principales: portada, proyectos, contacto.
3. Guarda esta información en un archivo notas_visuales.md.

# Parte 2: Encantando tus Pergaminos (Aplicación de CSS)

  Ahora, deberás aplicar estilos CSS externos a todo lo que has creado en los días anteriores:

## Archivos a mejorar:
1. pergamino.html (día 1)
2. proyectos.html (día 2 – formulario y horario)
3. Cualquier otra página adicional que hayas creado
## Instrucciones:

Crea un archivo css para cada proyecto y enlázalo a cada HTML con:
```HTML
<link rel="stylesheet" href="estilos.css">
```
  
# Aplica los siguientes encantamientos:
## Estilo general:
- Colores de fondo y texto (background-color, color)
- Tipografías elegidas con @import o `<link>` de Google Fonts
- Estilos para h1, h2, p, a, button, etc.
## Layout:
- Usa Flexbox para tu sección de redes o formulario.
- Usa Grid para tu tabla de horarios (¡que luzca como un calendario mágico!).
- Ajusta márgenes y paddings para que todo respire visualmente (margin, padding).
## Efectos mágicos:
- Transiciones suaves (transition)
- Hover en botones o enlaces (:hover)
- Bordes redondeados (border-radius)
- Animaciones simples con @keyframes (opcional, nivel avanzado)
## Diseño responsivo:
- Usa media queries para que tu sitio se adapte a pantallas pequeñas.
- Cambia tamaños de fuente y reorganiza elementos en celulares.

# Para aprobar esta práctica, tus documentos deben cumplir:

1. Tener tus documentos de estilos css correctamente enlazados.
    
2. Usar mínimo 3 secciones distintas estilizadas (portada, proyectos, contacto).
    
3. Aplicar colores personalizados y tipografías importadas.
    
4. Aplicar Flexbox o Grid en al menos una parte.
    
5. Ser legible en celular (puede usarse el modo inspección del navegador).
